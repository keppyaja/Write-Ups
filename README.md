# Write-Ups

## About

So, this is my repository to archive all of my the write up for some CTF Challenges that me/my team can solve. Alright, that's all from me.

## Archive Structure

- `pwn/` or `binex/` - Exploitation challenges and binary analysis write-ups.
- `web/` - Web application security challenges and web exploit write-ups.
- `crypto/` - Cryptography challenges and cipher analysis write-ups.
- `reversing/` - Reverse engineering and binary reversing write-ups.
- `forensics/` - Digital forensics challenges and forensic analysis write-ups.
- `misc/` - Miscellaneous challenges that don't fit other categories.

## Contribution

Add new write-ups by creating a new markdown file in the relevant directory and summarizing the challenge, approach, and solution.
